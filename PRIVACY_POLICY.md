# Privacy Policy

Last updated: May 13, 2026

## Overview

AI Token Optimizer is a browser extension that helps users optimize prompt text and estimate token usage.  
This extension is designed to work locally in the browser.

## Data We Process

The extension may process the following user-provided content:

- Prompt text entered in the popup
- Text selected by the user on supported AI websites
- Optimized output text generated from user input

This processing is used only to provide the extension's core functionality.

## Data Storage

The extension uses browser local storage (`chrome.storage.local`) to store:

- User preferences (for example, selected compression level)
- Recent extension state (for example, recent input/history used by extension features)

Data is stored locally on the user's device and is not sent to our servers.

## Data Sharing and Transfer

- We do **not** sell user data.
- We do **not** transfer user data to third parties for advertising or unrelated purposes.
- We do **not** use or transfer user data to determine creditworthiness or for lending purposes.

## Remote Code and External Processing

- The extension does **not** execute remote code.
- The extension does **not** send prompt text or user content to external services by default.

## Permissions Use

- `storage`: Save user preferences and local extension state.
- `clipboardWrite`: Copy optimized text when the user requests copy actions.
- `contextMenus`: Provide right-click actions for selected text.
- `notifications`: Show status feedback for context-menu actions.
- Host permissions: Run content scripts only on supported AI websites to provide in-page optimization features.

## User Control

Users can:

- Clear extension-managed local data by removing extension data or uninstalling the extension
- Avoid storing new local state by not using optional history features

## Changes to This Policy

We may update this policy from time to time. Updates will be reflected by changing the "Last updated" date above.
